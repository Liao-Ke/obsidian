用 SSH 暴露本地魔法的监听端口到服务器

```
ssh -R [远程端口]:localhost:[本地端口] 用户名@IP
```

然后服务器上设置环境变量

```
export http_proxy=http://localhost:[远程端口]
export https_proxy=http://localhost:[远程端口]
export all_proxy=socks5h://localhost:[远程端口]
```

端口这儿，看你的魔法是不是混合模式监听（一般应该都是）。  
不是的话，socks5h 换成 http 也行，或者再 -R 多暴露一个端口，或者不设置 all\_proxy 也行，大部分不会用到 all\_proxy。
